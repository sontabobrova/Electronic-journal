# Приемочные сценарии

## Базовый запуск

- Сервисы `backend`, `db`, `redis`, `celery-worker`, `celery-beat` запущены.
- `GET /health/` возвращает `{"status":"ok"}`.
- Команда `python manage.py create_demo_data` завершается успешно и повторный запуск не создает дубликаты.

## Авторизация и роли

- Администратор входит с `admin / admin123`.
- Преподаватель входит с `teacher / teacher123`.
- Студент входит с `student / student123`.
- После трех неверных попыток входа пользователь временно блокируется.
- Студент и преподаватель не получают доступ к административным API.

## Администратор

- Создает, редактирует и просматривает пользователей.
- Сбрасывает пароль пользователю.
- Управляет группами, дисциплинами, периодами, студентами, преподавателями и назначениями.
- Видит сводки `/api/admin-cabinet/dashboard/`, `/users-summary/`, `/education-summary/`, `/journal-summary/`.
- Видит журнал аудита `/api/audit/logs/`.

## Преподаватель

- Видит только свои назначения.
- Видит студентов групп, к которым привязан назначением.
- Создает работу в журнале оценок.
- Выставляет и изменяет оценку студенту своей группы.
- Создает занятие.
- Отмечает посещаемость по своему занятию.
- Получает напоминание о закрытии учебного периода за 3 дня до окончания.

## Студент

- Видит свой дашборд, средний балл и сводку посещаемости.
- Видит только свои оценки.
- Видит только свою посещаемость.
- Не может изменять оценки, занятия и посещаемость.

## Отчеты

- Администратор формирует отчет по успеваемости в `csv`, `xlsx`, `pdf`.
- Администратор формирует отчет по посещаемости в `csv`, `xlsx`, `pdf`.
- Преподаватель формирует отчеты только по своим данным.
- Студент не получает доступ к чужим отчетным данным.
- Сформированный файл доступен по `download_url`.

## Безопасность и аудит

- Все API, кроме healthcheck и login, требуют авторизации.
- Ролевые ограничения возвращают `403` при попытке доступа не к своей зоне ответственности.
- В ответах есть `X-Request-ID`.
- В аудите фиксируются входы, ошибки входа, выходы, изменения пользователей, оценки, посещаемость и отчеты.
